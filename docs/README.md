## 기능 구현

### 📃 시작 메시지 출력

### 📃 다리 길이 입력 기능 구현
```
다리길이 입력값 예외 체크 [3이상 20이하]
```

### 📃 이동할 칸 입력 기능 구현
```
칸 이동 입력값 예외 체크 [U or D]
```
### 📃 이동상태 출력기능 구현
```
맞으면
 [ O ]
 [   ]

틀리면
 [ X ]
 [   ]
``` 

### 📃 최종 게임 결과 출력 기능 구현

### 📃 게임 실패 시, 재시작 여부 입력 기능 구현
```
재시작 여부 입력 값 예외 체크 [R or Q]
```

### 📃 재시작 여부 처리 기능 구현
```
"R" 입력 시 다시 시작

"Q" 입력 시
 게임 성공 여부 출력
 총 시도한 횟수 출력
 게임 종료
```
