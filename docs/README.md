## 기능 구현
```
    - 시작
        - 시작 메시지 출력
        - "\n" 출력
        
    - 다리 생성
        - 다리길이 입력 메시지 출력
        - 다리길이 입력
            - 예외처리 : 입력값 범위 체크[3이상 20이하]
        - 다리길이를 통해 U,D로 구성된 다리 Collection 생성
        
    - 이동
        - 칸 이동 메시지 출력
        - 이동할 칸 입력(U or D)
            - 예외처리 : 입력값 체크 [U or D]
        - 화면에 이동 상태 출력
            - 맞으면
            [ O ]
            [   ]
            - 틀리면
            [ X ]
            [   ]
        - "\n" 출력
    
    - 게임종료
        - 재시작 여부 입력 메시지 출력
            - "R" 입력 시 재시도
                - "이동" 단계부터 다시 시작
            - "Q" 입력 시 종료
                - 게임 성공 여부 출력
                - 총 시도한 횟수 출력
    - 
```

## 실행 결과 예시
```
    다리 건너기 게임을 시작합니다.

    다리의 길이를 입력해주세요. 
    3
    
    이동할 칸을 선택해주세요. (위: U, 아래: D)
    U
    [ O ]
    [   ]
    
    이동할 칸을 선택해주세요. (위: U, 아래: D)
    U
    [ O | X ]
    [   |   ]
    
    게임을 다시 시도할지 여부를 입력해주세요. (재시도: R, 종료: Q)
    R
    이동할 칸을 선택해주세요. (위: U, 아래: D)
    U
    [ O ]
    [   ]
    
    이동할 칸을 선택해주세요. (위: U, 아래: D)
    D
    [ O |   ]
    [   | O ]
    
    이동할 칸을 선택해주세요. (위: U, 아래: D)
    D
    [ O |   |   ]
    [   | O | O ]
    
    최종 게임 결과
    [ O |   |   ]
    [   | O | O ]
    
    게임 성공 여부: 성공
    총 시도한 횟수: 2
```
